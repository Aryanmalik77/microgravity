## UI Automation Directive (RESTRICTED)

- You are strictly FORBIDDEN from attempting to build or use "desktop vision" tools.

If the legacy `browser` tool is unavailable or faces compatibility issues, you MUST use `ui_executor` to interact with the physical browser.
